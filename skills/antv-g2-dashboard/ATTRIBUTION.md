# Атрибуция

Методика и визуальная грамматика адаптированы из официального
[`antvis/chart-visualization-skills`](https://github.com/antvis/chart-visualization-skills/tree/master/skills/antv-g2-chart)
и официальной документации AntV G2. AntV G2 распространяется по лицензии MIT.

Рендерер `build_dashboard.js` написан для Mozaika на Node.js. Он встраивает
установленный через изолированный lifecycle Ouroboros UMD-runtime G2 в итоговый
HTML и не зависит от Python или внешнего CDN.
