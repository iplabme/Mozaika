СЛАЙД 01
От аналитика к создателю агентов

Тезис слайда

ИИ превращает ML/DS-специалистов из потребителей аналитики в архитекторов автономных систем.

Ключевые цифры

  • Prompt engineering: 3.17% топ API-запросов
  • Agent architecture: 0.61% растущий запрос
  • AI-автономия Data Scientists: 3.1 из 4

Связь с аудиторией

Профиль ML/DS-специалиста проходит структурную трансформацию. Это влияет на найм, обучение и проектирование команд.

Эволюция роли

Переход

Что показать

СЛАЙД 06
Два режима работы с AI

Data Scientists — усилители

Augmentation 52.6% — Data Scientists чаще используют AI для расширения своих возможностей, а не для замены рутины.

Systems Analysts — автоматизаторы

Automation 83.7% — Computer Systems Analysts работают в режиме автоматизации: AI выполняет задачи за них.

Вывод

Роли разделились: одни усиливают человеческий интеллект, другие автоматизируют повторяющиеся процессы.

СЛАЙД 07
Что создают Data Scientists

Топ-3 артефакта

  • Визуализации: 26.5% — лидер среди всех ролей
  • Скрипты и код: 15.6%
  • Презентации: 15.3%

Что это значит

Data Scientists создают визуальные продукты и презентации чаще, чем код. Это сдвиг от программирования к коммуникации инсайтов.

Сравнение

Software Developers создают code_fix_or_debug в 38.9% случаев. Data Scientists — лишь 1.9%. Профиль смещён.

Топ-5 запросов

  • Workflow state routing: 3.05%
  • Structured data output: 2.31%
  • Agent architecture: 0.61%

Ключевой сигнал

Prompt engineering и Agent architecture в топе API-запросов — специалисты проектируют промпты и архитектуру агентов, а не просто ищут информацию.

Трансформация

Запросы смещаются от поиска информации к проектированию автономных систем. Это — практический след эволюции роли.

СЛАЙД 09
Автономия и новые компетенции

AI-автономия

Data Scientists: 3.1 из 4 — наивысший показатель среди ML/DS-ролей. Это значит, что AI берёт на себя значительную часть работы.

Новые компетенции

  • Prompt engineering (3.17%)
  • Agent architecture (0.61%)
  • Structured data output (2.31%)

Традиционные задачи

Поиск информации и программирование остаются, но дополняются проектированием агентов и структурированным выводом данных.

СЛАЙД 10
Вывод: от аналитика к архитектору

Главная мысль

Профиль ML/DS-специалиста эволюционирует от потребителя аналитики к создателю автономных систем.

Три доказательства

  • Наивысшая AI-автономия (3.1/4) среди всех ролей
  • Augmentation > Automation (52.6% vs 47.4%)
  • Prompt engineering и Agent architecture в топе API-запросов

Практическое следствие

Командам нужны специалисты, способные проектировать агентов, а не только анализировать данные. Это меняет требования к найму и развитию.

Источники

Claude Chat (1.6M строк) + 1P API (492K строк), апрель–май 2026. 23 ML/DS-роли, 51 метрика, 6 разрезов.
