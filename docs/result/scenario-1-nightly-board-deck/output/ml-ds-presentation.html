Вывод: от аналитика к архитектору

Профиль ML/DS-специалиста эволюционирует. Data Scientists — на передовой этого сдвига: наивысшая AI-автономия, доминирование augmentation над automation, и выход prompt engineering и agent architecture в топ API-запросов. Специалисты переходят от потребления аналитики к созданию автономных систем. Источники: Claude Chat (1.6M строк) + 1P API (492K строк), апрель–май 2026.
